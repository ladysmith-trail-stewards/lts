# Ladysmith Trail Stewards Website

A clean, responsive website for the Ladysmith Trail Stewards organization in Ladysmith, BC, Canada.

## Overview

This is a static website built with HTML, CSS, and Tailwind CSS that serves as the online presence for the Ladysmith Trail Stewards - a volunteer-driven organization committed to planning, maintaining, and advocating for sustainable trails in the Ladysmith area.

## Features

- **Responsive Design**: Mobile-first design that works on all device sizes
- **Modern Styling**: Uses Tailwind CSS for clean, professional appearance
- **Contact Integration**: 
  - Contact form powered by Formspree
  - MailerLite email signup integration
  - Facebook Group integration
- **Image Gallery**: Showcases trail and forest imagery
- **Charter Page**: Detailed organizational charter and mission statement

## Development

To modify this website:

1. Edit the HTML files directly for content changes
2. Tailwind CSS classes are used throughout - refer to [Tailwind documentation](https://tailwindcss.com/docs)
3. Images should be optimized for web (JPEG format, reasonable file sizes)
4. Test responsiveness across different screen sizes
